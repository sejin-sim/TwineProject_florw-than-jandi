# TwineProject_flower-than-jandi

## 2020-1학기 숭실대 디지털스토리텔링(인터렉티브 스토리텔링) Twine 프로젝트

### * **프로젝트명 : 꽃보다 잔디**

&nbsp;&nbsp;&nbsp; 1. 서비스 설명 : 추억의 드라마 '꽃보다 남자'의 금잔디가 되어 '꽃보다 잔디'에서 새로운 스토리를 만들어 보자    
&nbsp;&nbsp;&nbsp; 2. 타겟 : 원작을 시청했던 ‘20~30대 여성’   
&nbsp;&nbsp;&nbsp; 3. 스토리 : 총 33개 패시지. 1회 진행 시 21개(12개 : 미선택) 패시지 실행 가능. 총 6번의 선택을 통해 결말 4개 中 1개가 채택
